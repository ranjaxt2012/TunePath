# Harmonium Sargam Samples

Add one MP3 file per sargam note for BPM-driven notation playback:

- `Sa.mp3`
- `Re.mp3`
- `Ga.mp3`
- `Ma.mp3`
- `Pa.mp3`
- `Dha.mp3`
- `Ni.mp3`

Each file should be a short (~0.5–1.5s) harmonium note. The app will play them in sequence based on the lesson notation and BPM.
